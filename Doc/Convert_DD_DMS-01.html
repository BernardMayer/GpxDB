Convertisseur de coordonnées GPS

Cet outil est dédié à la conversion de coordonnées GPS. Dès que vous modifiez un des champs de saisie, (soit dans les coordonnées décimales, soit dans les coordonnées sexagésimales), tous les autres champs sont automatiquement mis à jour, de même que la position du marqueur sur la carte.

Les coordonnées GPS sont aussi affichées dans la fenêtre d'information sur la carte, dans un format clair et facile à copier coller.

Vous pouvez également commencer votre conversion de coordonnées GPS en cliquant sur la carte, ce qui aura pour effet de pré-remplir les champs avec les coordonnées GPS du point cliqué. Dans tous les cas, l'adresse ne sera pas décodée automatiquement. Si vous souhaitez convertir des coordonnées GPS en adresse, vous devez cliquer sur le bouton "Obtenir l'adresse" sous les coordonnées GPS décimales.

DD (degrés décimaux)*

DMS (degrés, minutes, secondes)*

* Système géodésique WGS 84
